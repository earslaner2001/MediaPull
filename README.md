Twitter / X ve YouTube bağlantılarından medya indiren Windows masaüstü uygulaması.

---

## Özellikler

- **Twitter / X** — Tweet veya video bağlantısından mümkün olan en iyi kalite
- **YouTube** — Link doğrulama, otomatik dosya adı, format seçimi (1080p, MP3, vb.)
- **Kayıt yeri** — `İndirilenler\MediaPullDownloads`
- İlk çalıştırmada `yt-dlp` ve `ffmpeg` otomatik hazırlanır

## Kurulum

1. [Releases](https://github.com/earslaner2001/Arslaner-Download-Express/releases) sayfasından Windows yükleyicisini indirin.
2. Kurulumu tamamlayıp uygulamayı başlatın.

### Geliştirici derlemesi

```bash
npm install
npm start
```

Kurulum paketi (NSIS):

```bash
build-setup.bat
```

Çıktı: `dist/`
